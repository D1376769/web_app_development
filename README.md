## 系統簡介

本系統提供讀書筆記紀錄與整理功能，幫助學生有效管理學習內容，提升複習效率與學習成效。

---

## 技術棧

- **前端：** Jinja2
- **後端：** Python + Flask
- **資料庫：** SQLite
